# 🎓 Student Management System (C++)

## 📌 Description
A console-based application developed in C++ to manage student records efficiently using file handling techniques.

## 🚀 Features
- Add new student records
- View all students
- Search student by roll number
- File storage using binary file handling
- Menu-driven user interface

## 🛠️ Technologies Used
- C++
- File Handling
- Object-Oriented Programming (OOP)

## 💡 How It Works
The system stores student data in a file and allows users to perform operations like add, search, and view records through a simple menu interface.
